Terminal: tldr

May 8, 2019
CLI Terminal TOW

This week on tip of the week is on the cool terminal application called tldr.

What is tldr?

This is a simple tool that allows you to look up commands in a simple way, think man pages but more accessible. Although nothing beats the sheer depth of the inbuilt man pages, the tldr application has something even greater. It supports help pages for non-unix-native apps. This in itself makes tldr something interesting. Without wasting time lets get down to some show and tell code.

The setup:

There are two methods to try this tool out the easy least effort way would be to use the website client. What you are presented is something very simple a search box and some general explaination on how best to use the tool.

The other method of accessing this tool is via installing a npm package npm install -g tldr. This brings the tool right into your terminal.

How to use:

For those who have installed it locally, the command to remember is 

 tldr [command]
So for example, punching 
 tldr aws
on the terminal would result in the lovely screenshot below

The above shows how simple the instructions are and although tailored toward native terminal apps like sed, awk and tail, it is a good starting point for those who are just starting with terminal and are not ready to dive deep into the man pages.
